# Student Portal

The Student Portal provides students and parents/guardians with access to scheduling, resources, homework, and communication with their tutor.

## Core Features

### Overview & Information
- [Dashboard](./dashboard.md) - Overview of upcoming sessions, recent activity, progress
- [Contact Info](./contact-info.md) - Student and family contact information, emergency contacts

### Scheduling
- [Calendar](./calendar.md) - View scheduled sessions, upcoming events, availability

### Educational Resources
- [Borrowing Library](./borrowing-library.md) - View borrowed materials, due dates, history
- [Online Resources](./online-resources.md) - Access resources shared by tutor
- [Homework Submission](./homework-submission.md) - Submit assignments, view feedback

### Session Information
- [Attendance and Notes](./attendance-notes.md) - View session attendance, tutor notes, session summaries

### Communication
- [News & Message History](./news-message-history.md) - Communication history, announcements from tutor

---

**Note:** All features should provide appropriate access levels for both students and parents/guardians, with parental controls where necessary.
